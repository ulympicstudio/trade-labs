## Scoring Algorithm

### Catalyst Score (0-100)
```
For each signal:
  weight = catalyst_type_multiplier Ã— source_credibility Ã— signal_confidence
  contribution = weight Ã— direction Ã— magnitude
  
Aggregate = (sum of contributions / total weight) Ã— 25 + 50
Normalized to 0-100
```

### Combined Score (0-100)
```
Combined = (Catalyst Score Ã— 0.60) + (Technical Score Ã— 0.40)

âœ“ This is the OPPOSITE of your old scanner-heavy approach
âœ“ Makes sense: catalysts are rare, technical setups are common
```

### Trade Decision Thresholds
- **Minimum Combined Score**: 70.0
- **Minimum Confidence**: 55%
- **Minimum Urgency**: 50% (reject stale news)
